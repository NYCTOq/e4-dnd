1. Bu klasörün içindeki package.json, package-lock.json ve src klasörünü proje köküne kopyalayın.
2. Windows sorarsa "Hedefteki dosyaları değiştir" seçeneğini kullanın.
3. Proje kökünde çalıştırın:
   npm.cmd install
   npm.cmd test
   npm.cmd run build
