# SRD Attribution

E4 D&D uses class names, rules references, and structured game data derived from the official **System Reference Document 5.1** and **System Reference Document 5.2.1**, made available under the **Creative Commons Attribution 4.0 International license (CC BY 4.0)**.

The project uses concise, original summaries and structured progression/runtime data rather than reproducing full book text. Closed or non-open sourcebook content is not bundled as default data. Users may represent unsupported options through the local Homebrew layer.

## Project policy

- Keep 2014 and 2024 ruleset data separated.
- Prefer original mechanical summaries over copied prose.
- Preserve source and attribution metadata in exported ruleset packages.
- Treat content outside the open SRD scope as user-created Homebrew rather than bundled default content.
