# RC2 Release Checklist

## Otomatik blocker kapıları

- [ ] RC2 metadata audit geçti
- [ ] Technical debt audit geçti
- [ ] ESLint geçti
- [ ] Kritik unit suite geçti
- [ ] Production build geçti
- [ ] SHA-256 artifact manifest üretildi
- [ ] RC1 kritik rota smoke geçti
- [ ] RC2 çapraz özellik regresyonu geçti
- [ ] Browser pageerror ve console error listesi boş

## Manuel son prova

- [ ] Yeni karakter oluştur
- [ ] Karakteri bir seviye yükselt
- [ ] Character Sheet aç
- [ ] Inventory & Economy panelini kontrol et
- [ ] Yeni combat oluştur
- [ ] Backup dosyası seçip güvenlik önizlemesini kontrol et
- [ ] Mobil görünümde yatay taşma olmadığını doğrula
- [ ] Uygulamayı kapatıp yeniden aç, kayıtların kaldığını doğrula

Herhangi bir blocker varsa stable sürüm ertelenir. Kullanıcıyı ücretsiz QA departmanı yapmak yasaktır.
