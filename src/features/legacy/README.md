# LegacyApp

Bu dosya eski tek parça App.tsx içeriğini geçici olarak taşır.
Amaç App.tsx dosyasını inceltip sonraki paketlerde feature feature güvenli biçimde parçalamak.

Sıradaki refactor paketlerinde buradan şu alanlar çıkarılacak:
- dashboard
- characters
- builder
- spellbook
- inventory
- campaigns
- homebrew
- dice
- backup
- library
- play-mode
