# Single-Class Builder UI Guardrails v5.72

Bu paket v5.71 tek-sınıf veri sertifikasyonunu Builder Review ekranına bağlar.

## Kapsam
- 2014 ve 2024 resmî ruleset ayrımı
- Race/Species ve class zorunluluk kontrolü
- Class'a ait subclass açılma seviyesinin UI'da doğrulanması
- Subclass seviyesi gelmeden seçim zorunluluğu üretmeme
- Subclass seviyesi geldiğinde eksik seçimi açıkça gösterme
- Homebrew karakterleri resmî sertifikasyon kapsamından ayrı gösterme

## Uygulama
Paket klasörünün içeriğini proje köküne kopyalayın ve hedef dosyaları değiştirin.

```powershell
npm.cmd install
npm.cmd test
npm.cmd run build
```
