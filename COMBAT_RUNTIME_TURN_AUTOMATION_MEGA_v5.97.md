# E4 D&D v5.97 Combat Runtime & Turn Automation Mega Pack

## Kapsam

- Aktif savaşçı bazlı Action, Bonus Action ve Reaction ekonomisi
- Hareket kullanımının speed değerinde sınırlandırılması
- Yeni turn başında aktif savaşçının ekonomisinin sıfırlanması
- Yeni round başında tüm reaction haklarının yenilenmesi
- Concentration hasarında otomatik save DC üretimi
- Death Save başarı ve başarısızlık sayaçları
- Natural 1 için iki başarısızlık
- Natural 20 için otomatik stabilizasyon
- Üç başarıda stabilized, üç başarısızlıkta dead sonucu
- Combat Tracker içinde mobil uyumlu Turn Automation paneli
- Unit ve Playwright E2E doğrulaması

## Doğrulama

```powershell
npm.cmd install
npm.cmd run verify:combat-runtime
```
