E4 D&D Races / Species + Backgrounds V1

1. Bu klasörün içeriğini D:\Projects\e4_dnd içine çıkarın.
2. Mevcut dosyaların üzerine yazılmasına izin verin.
3. PowerShell:
   cd D:\Projects\e4_dnd
   npm.cmd install
   npm.cmd run check
4. Kontrol geçerse:
   git add -A
   git commit -m "feat: add races species and backgrounds"
   git push

Not: public/data altındaki backgrounds.json dosyaları zorunludur. GitHub Pages veri yolları Vite BASE_URL ile uyumludur.
